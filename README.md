# Study Planner

## What is this project?
This is a simple web application to help students plan their daily study.

## Why I made this?
Many students get confused during exams and waste time.
This project helps in better time management.

## Features
- Add subjects
- Plan daily study time
- Simple and clean interface

## Technologies Used
- React
- JavaScript
- HTML
- CSS
